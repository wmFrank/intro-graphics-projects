### 实验思路
- 构造绳子
首先根据起始位置以及质点个数给创建质点并添加到masses数组中，然后根据pinnned_nodes来确定哪些质点是固定点。然后每两个质点创建一根弹簧，添加到springs数组中。
- 显式欧拉法
首先对springs数组进行遍历，使用胡克定律计算每个质点受到的弹力（注意是计算累加弹力）。然后对masses数组进行遍历，对于其中非固定的点，累加弹力和重力计算加速度，然后使用显式欧拉公式计算每个质点的下一个位置和速度。
- 半影式欧拉法
和显式欧拉法唯一不同就是计算每个质点下一个位置时用到的是下一个速度。
- 显式Verlet
首先遍历springs数组，根据每根弹簧的last_position和当前的position依据恢复到弹簧原长这个前提进行修正。然后遍历masses数组，对于其中非固定点，求取加速度(此时不考虑弹簧内力，仅考虑外力，在此实验中外力只有重力)。求取加速度后应用Verlet的公式计算下一个位置，并且将当前的位置赋值给last_position。
- 阻尼
对于欧拉法来说，添加阻尼的方法是对于某一个质点需要添加一个逆速度方向的衰减，我是把这个衰减看成一种阻力，加到这个质点受力里面去了。对于Verlet法来说，添加阻尼的方法就直接按照提供的公式，对于某个质点移动的距离进行一个衰减。
